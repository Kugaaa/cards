# 闪卡工具 (Flashcards)

一个简洁美观的闪卡学习工具，支持填空题和问答题两种模式。

## 功能特点

- 📥 **多种导入方式**：支持本地 JSON 文件上传或 URL 远程获取
- 🔄 **卡片翻转效果**：点击卡片查看答案，动效流畅
- 📝 **两种题型支持**：
  - 填空题：题干中使用 `$$答案$$` 标记答案位置
  - 问答题：正面显示问题，背面显示答案
- 🎨 **三套配色主题**：玫瑰粉、薄荷绿、薰衣草
- ⌨️ **键盘快捷键**：
  - `←` `↑`：上一张
  - `→` `↓`：下一张
  - `Space`：翻转卡片

## 数据格式

JSON 数组格式：

```json
[
  {
    "题干": "React 中，$$答案$$ 用于管理组件状态。",
    "答案": ["useState"]
  },
  {
    "题干": "什么是虚拟 DOM？",
    "答案": ["虚拟 DOM 是真实 DOM 的 JavaScript 对象表示..."]
  }
]
```

### 填空题

题干中使用 `$$答案$$` 标记答案位置，支持多个填空：

```json
{
  "题干": "$$答案$$ 和 $$答案$$ 是 React 最常用的两个 Hook。",
  "答案": ["useState", "useEffect"]
}
```

### 问答题

题干中不包含 `$$答案$$` 标记，翻转后直接显示答案列表：

```json
{
  "题干": "解释一下闭包是什么？",
  "答案": [
    "闭包是指函数能够访问其外部作用域的变量...",
    "示例代码..."
  ]
}
```

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```
